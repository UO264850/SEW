Diferencias con Javascript

Una comparación típica

Ya que Typescript está fuertemente isnpirado en Javascript, es muy común que se compare entre ambas opciones a la hora de decidir cual es mejor.

Comparación

Característica Typescript Javascript
¿Orientado a objetos? Si, orientado a objetos No, orientado a prototipos
¿Soporta static typing? Si, soporta static typing No soporta static typing
¿Soporta interfaces? Si, soporta interfaces No, no soporta interfaces
¿Compilado o interpretado? Compilado Interpretado
¿Cuál es más rápido? Compila más lento Compila más rápido
Modularización Ofrece soporte directo para módulos Ofrece soporte para módulos a través de EcmaScript 6
Decoradores Ofrece soporte para decoradores No ofrece soporte para decoradores
Adecuación a proyectos Ventajoso en proyectos medianos o grandes Ventajoso en proyectos pequeños
